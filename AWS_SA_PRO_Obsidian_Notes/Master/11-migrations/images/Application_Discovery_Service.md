Application Discovery Service
------------------------------

### [[RDS_Instance_Types|1. Advanced Architecture]]

Application Discovery Service (ADS) is a fully managed service that helps plan application migrations by collecting data about applications running in their current environment. ADS collects data using two methods: Server Migration Connector (SMC) or Lightweight Directory Access Protocol (LDAP) server.
