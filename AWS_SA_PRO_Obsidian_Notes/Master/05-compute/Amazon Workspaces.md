
Amazon WorkSpaces is a managed, secure Desktop-as-a-Service (DaaS) solution. It supports both Windows and Linux desktops, ensuring a seamless transition for users.

The integration with on-premises Active Directory via an AD Connector allows employees to use their existing credentials. The use of a RADIUS server for [[mfa]] adds an extra layer of [[appsync|security]], ensuring that the remote desktop experience is both secure and familiar to the users.
